# 🎨 Theme Manager para Antigravity

¡Lleva la personalización visual de tu IDE al siguiente nivel! **Theme Manager** es la extensión definitiva para gestionar, descubrir y modificar temas, iconos y la interfaz de Antigravity directamente desde un elegante panel integrado en tu barra lateral.

---

## ✨ Características Principales

### 🌈 Gestión Instantánea de Temas
Descubre y cambia entre todos los temas instalados en tu IDE sin tener que navegar por menús ocultos. **Haz clic en cualquier tema y se aplicará al instante**. ¡Prueba todos los temas que quieras en segundos!
- 🏷️ **Etiquetas Personalizadas:** Organiza tus temas favoritos añadiendo etiquetas (tags) para filtrarlos rápidamente.
- ⭐ **Favoritos:** Marca los temas que más te gustan para tenerlos siempre a la mano.
- 🎲 **Botón Aleatorio:** ¿No sabes qué tema usar hoy? Deja que el botón aleatorio elija por ti.

### 🖼️ Fondos de Pantalla Integrados
¿Cansado del fondo gris aburrido? Selecciona cualquier imagen de tu computadora (`PNG`, `JPG`, `GIF`, `WebP`) y conviértela en el fondo de tu editor.
- Ajusta la opacidad con un deslizador en vivo.
- **Inyección Segura:** Nuestro *"Interruptor Maestro"* te permite activar y desactivar estas modificaciones de CSS de forma totalmente segura sin romper el editor.

### 🔧 Personalización UI en Vivo
Ajusta la interfaz de tu editor sin tocar un solo archivo JSON de configuración. Los cambios reaccionan al instante:
- Tamaño de fuente del editor y la terminal.
- Nivel de Zoom general de la interfaz.
- Tamaño de las barras de desplazamiento (Scrollbars).
- Ocultar/Mostrar la Barra de Estado, Minimapa y Barra de Actividad.

### ✏️ Creador de Temas (Theme Creator)
¿No encuentras el tema perfecto? ¡Créalo tú mismo!
- Toma cualquier tema existente como plantilla usando la función **"Copiar y editar"**.
- Ajusta los colores principales del editor, la barra lateral, botones y más.
- Guarda tu creación y úsala permanentemente.

---

## 🚀 Instalación y Uso

1. Instala la extensión desde el Marketplace de Antigravity o desde un archivo `.vsix`.
2. Una vez instalada, aparecerá un nuevo ícono 🎨 en tu barra de actividad izquierda.
3. Haz clic en el ícono para abrir el panel completo y empezar a personalizar tu entorno.

> **Nota Técnica sobre Fondos:** La función de añadir una imagen de fondo inyecta código CSS directamente en el archivo principal del editor (`workbench.desktop.main.css`). Si alguna vez deseas desactivarlo o limpiar la inyección, simplemente apaga el **Interruptor Maestro** dentro de la pestaña "Fondo". La extensión limpiará automáticamente el código al cerrarse o desactivarse.

---

## 🛠️ Requisitos
- **Antigravity IDE** o cualquier editor basado en VS Code compatible (`^1.74.0`).
- No requiere dependencias externas.
